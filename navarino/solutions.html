So, the query to get the maximum salary from each department with the employee name is:

SELECT employee, MAX(salary), Department.department FROM General INNER JOIN Department ON General.department_id = Department.id GROUP BY department;



In this query I m selecting the employee column, the maximum value of the salary column from GENERAL TABLE along with the department column from Department table, I create an Inner Join with the Department table where I assign the department_id column equal to the value of the primary key of the Department table and Group the result by department.
